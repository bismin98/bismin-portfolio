# Portfolio Bismin

Website portfolio colorfull dengan tema fotografer, desain grafis, dan programmer.

## Menjalankan Proyek

- Install dependency: `npm install`
- Jalankan dev server: `npm run dev`

## Catatan

- Ganti placeholder foto di `public/portrait-placeholder.svg` dengan foto asli.
- Sesuaikan teks dan daftar project di setiap halaman.
